**On this page**, you will need to introduce your worklog. **How** did you complete it? How many weeks did you take to complete the program? **What** did you do in those weeks?

Typically, and as a standard, a worklog is carried out over about 3 months (throughout the internship period) with weekly contents as follows:

**Week 1:** [Become familiar with the First Cloud AI Journey program, create an AWS account, and practice basic AWS Budgets and AWS Support concepts.](1.1-week1/)

**Week 2:** [Study IAM, VPC, EC2, and how to assign permissions to EC2 instances using IAM Roles.](1.2-week2/)

**Week 3:** [Learn AWS Cloud9 and practice deploying a static website using Amazon S3 Static Website Hosting.](1.3-week3/)

**Week 4:** [Study AWS infrastructure services such as RDS, DynamoDB, ElastiCache, Auto Scaling, Route 53, CloudFront, Lightsail, and data migration services.](1.4-week4/)

**Week 5:** [Learn Lambda, Systems Manager, Session Manager, AWS CDK, API Gateway, and start defining the idea for the hospital management project.](1.5-week5/)

**Week 6:** [Study KMS, Secrets Manager, and Cognito, while analyzing hospital business requirements and designing the database for the MeChain AI project.](1.6-week6/)

**Week 7:** [Learn SQS, SNS, Firewall Manager, and messaging, queueing, and security mechanisms in AWS application architecture.](1.7-week7/)

**Week 8:** [Design the overall AWS architecture for the project, configure VPC-related components, build the web/mobile interface, and adjust the database design.](1.8-week8/)

**Week 9:** [Review the AWS architecture of MeChain AI and define the flows for Cognito, Lambda, API Gateway, DynamoDB, S3, CloudFront, AMB, and VNPay/MoMo payment.](1.9-week9/)

**Week 10:** [Set up AWS infrastructure using CDK, configure Cognito, Lambda, API Gateway, DynamoDB, S3, and CloudFront, and deploy the frontend/backend to AWS.](1.10-week10/)

**Week 11:** [Standardize CCCD as the medical record ID, synchronize medical data, build the Medical Integrity Ledger, and design AMB and real VNPay/MoMo payment flows.](1.11-week11/)

**Week 12:** [Perform final deployment testing, reset the dataset, backfill the ledger, verify blockchain integrity, review AMB transactionHash, payment callbacks, and complete the report.](1.12-week12/)
