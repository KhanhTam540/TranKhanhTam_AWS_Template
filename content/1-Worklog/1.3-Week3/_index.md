* Explore cloud-based application development environments and understand modern hosting solutions for web applications.
* Gain hands-on experience in cloud code editing, terminal management, and cost-effective, serverless web distribution.

### Week 3 Achievements:
* Mastered cloud-based application development using AWS Cloud9, eliminating local environment dependencies and leveraging pre-installed SDKs alongside an integrated, pre-authenticated terminal.

* Successfully established a streamlined workflow by running, debugging, and managing code repositories seamlessly from a standard web browser interface.

* Developed proficiency in serverless web infrastructure by implementing Static Website Hosting using Amazon S3, enabling highly scalable and cost-effective web content distribution without the need for traditional server maintenance.

* Acquired deep knowledge of cloud storage security, correctly implementing S3 Bucket Policies and access controls to safely expose public front-end interfaces while maintaining proper data segregation.
