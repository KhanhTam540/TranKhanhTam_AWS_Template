### Week 5 Achievements:

* Gained a solid conceptual understanding of the internal mechanics and core architectures powering event-driven serverless functions, centralized host management, and code-defined server infrastructure (IaC).

* Developed primary hands-on familiarity with cloud routing services by building basic Amazon API Gateway structures capable of invoking decoupled AWS Lambda compute operations.

* Mastered fundamental interactions with AWS Systems Manager (SSM) modules by securely isolating parameters inside Parameter Store and utilizing Session Manager for direct browser terminal inputs without regular SSH keys.

* Attained a clear introduction to the AWS CDK infrastructure lifecycle, acquiring the ability to map, synthesize, and launch simple cloud components using high-level code syntax instead of manually adjusting the AWS Console.

* Strengthened the ability to concurrently utilize the AWS Web Console and command-line interfaces (CLI) to construct and inspect basic states of the studied serverless and configuration suites.
