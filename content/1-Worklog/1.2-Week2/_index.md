### Week 2 Objectives:

* Understand core cloud architecture components: Identity Management, Virtual Networking, and Compute Resources.
* Practice deploying secure, isolated network environments and configuring virtual servers with secure service permissions.

### Week 2 Achievements:
* Gained a thorough understanding of AWS IAM and mastered granular access control by implementing Multi-Factor Authentication (MFA) and the Least Privilege principle.

* Successfully designed and constructed an isolated virtual network environment using Amazon VPC, effectively managing inbound and outbound traffic through Internet Gateways, Security Groups, and stateless Network ACLs.

* Developed proficiency in deploying and managing virtual computing power on Amazon EC2 across different operating systems, including configuration of key pairs and EBS block storage.

* Mastered the implementation of Instance Profiles and IAM Roles for EC2, ensuring secure application interaction between virtual servers and non-compute services without exposing long-term AWS access credentials.
