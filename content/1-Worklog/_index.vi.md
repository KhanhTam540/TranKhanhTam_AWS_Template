**Trong trang này** bạn sẽ cần giới thiệu worklog của bạn **như thế nào**? Bạn hoàn thành chương trình trong vòng **bao nhiêu tuần**? Bạn **đã làm gì** trong các tuần đó?


Thông thường và cũng là tiêu chuẩn, một worklog được thực hiện trong khoảng 3 tháng (trong suốt thời gian thực tập) với nội dung các tuần như sau:

**Tuần 1:** [Làm quen với chương trình First Cloud AI Journey, tạo tài khoản AWS và thực hành các nội dung AWS Budgets, AWS Support cơ bản.](1.1-week1/)

**Tuần 2:** [Tìm hiểu IAM, VPC, EC2 và cách phân quyền bằng IAM Role cho máy chủ EC2 trong môi trường AWS.](1.2-week2/)

**Tuần 3:** [Tìm hiểu AWS Cloud9 và thực hành triển khai website tĩnh bằng Amazon S3 Static Website Hosting.](1.3-week3/)

**Tuần 4:** [Tìm hiểu các dịch vụ hạ tầng AWS như RDS, DynamoDB, ElastiCache, Auto Scaling, Route 53, CloudFront, Lightsail và các dịch vụ di chuyển dữ liệu.](1.4-week4/)

**Tuần 5:** [Tìm hiểu Lambda, Systems Manager, Session Manager, AWS CDK, API Gateway và bắt đầu lên ý tưởng cho đề tài dự án bệnh viện.](1.5-week5/)

**Tuần 6:** [Tìm hiểu KMS, Secrets Manager, Cognito, đồng thời phân tích nghiệp vụ và thiết kế cơ sở dữ liệu cho dự án MeChain AI.](1.6-week6/)

**Tuần 7:** [Tìm hiểu SQS, SNS, Firewall Manager và các cơ chế truyền tin, hàng đợi, bảo mật trong kiến trúc ứng dụng AWS.](1.7-week7/)

**Tuần 8:** [Thiết kế kiến trúc tổng thể dự án theo mô hình AWS, cấu hình VPC, xây dựng giao diện web/app và điều chỉnh cơ sở dữ liệu.](1.8-week8/)

**Tuần 9:** [Rà soát kiến trúc AWS của MeChain AI, xác định luồng Cognito, Lambda, API Gateway, DynamoDB, S3, CloudFront, AMB và thanh toán VNPay/MoMo.](1.9-week9/)

**Tuần 10:** [Thiết lập hạ tầng AWS bằng CDK, cấu hình Cognito, Lambda, API Gateway, DynamoDB, S3, CloudFront và triển khai frontend/backend lên AWS.](1.10-week10/)

**Tuần 11:** [Chuẩn hóa CCCD làm mã hồ sơ bệnh án, đồng bộ dữ liệu y tế, xây dựng Medical Integrity Ledger, thiết kế AMB và luồng thanh toán thật VNPay/MoMo.](1.11-week11/)

**Tuần 12:** [Kiểm thử triển khai cuối, reset dataset, backfill ledger, kiểm tra blockchain, rà soát AMB transactionHash, callback thanh toán và hoàn thiện báo cáo.](1.12-week12/)
